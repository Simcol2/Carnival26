Promo Codes
EmailJS
Public Key
Your EmailJS public key identifies your account. Find it at dashboard.emailjs.com → Account → API Keys. It typically starts with a short alphanumeric string. This is safe to expose client-side.
Service ID
The Service ID links to your email provider connection (Gmail, Outlook, etc.). Find it at dashboard.emailjs.com → Email Services. It looks like service_xxxxxxx.
Template ID
The Template ID refers to the confirmation email template sent to registrants. Found at dashboard.emailjs.com → Email Templates. Looks like template_xxxxxxx.
Formspree
Form Endpoint URL
Formspree receives a copy of every registration submission and can forward it to your email. Log in at formspree.io, create a new form, and copy the endpoint URL — it will look like https://formspree.io/f/xxxxxxxx.
Google Sheets
Web App URL
This is a Google Apps Script Web App URL that appends rows to a Google Sheet on each submission. In your spreadsheet open Extensions → Apps Script, paste your logging script, deploy as a Web App (access: Anyone), and copy the URL. Starts with https://script.google.com/macros/s/.
PDF Lookbook
PDF URL
URL of the costume lookbook PDF shown in the registration form's "View Costumes" modal. Host it on your domain or any CORS-friendly storage (e.g. GitHub Pages, Cloudflare R2). Must be publicly accessible and served with permissive CORS headers so PDF.js can load it.

Reset to Defaults?

This will clear all saved configuration from localStorage and restore the original default values. Images stored separately (logos, costume photos, etc.) will also be removed.

This action cannot be undone.
